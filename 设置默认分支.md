# 设置 GitHub 默认分支为 main

由于 GitHub 不允许删除当前默认分支，需要先在网页上更改默认分支设置。

## 操作步骤

### 方法 1：通过 GitHub 网页设置（推荐）

1. **访问仓库设置页面**
   - 打开：https://github.com/weoifgiljb/smart-dashboard/settings
   - 或：仓库页面 → Settings → Branches

2. **更改默认分支**
   - 在 "Default branch" 部分
   - 点击分支名称旁边的切换图标（或 "Switch to another branch"）
   - 选择 `main` 分支
   - 点击 "Update" 确认
   - 在弹出的确认对话框中点击 "I understand, update the default branch"

3. **删除旧的 master 分支**（可选）
   - 在仓库页面点击 "branches" 或访问：
     https://github.com/weoifgiljb/smart-dashboard/branches
   - 找到 `master` 分支，点击右侧的删除图标
   - 确认删除

### 方法 2：使用 GitHub CLI（如果已安装）

```bash
# 设置默认分支为 main
gh api repos/weoifgiljb/smart-dashboard -X PATCH -f default_branch=main

# 删除 master 分支
gh api repos/weoifgiljb/smart-dashboard/git/refs/heads/master -X DELETE
```

## 完成后的验证

设置完成后，可以运行以下命令删除远程 master 分支：

```bash
git push origin --delete master
```

或者直接访问仓库，确认默认分支已更改为 `main`。
