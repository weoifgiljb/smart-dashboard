# 蓝湖 AI UI 优化提示词 (精简版)

本文档汇集了基于项目需求的高密度 AI 提示词，请直接复制到蓝湖/MasterGo AI 中使用。

---

## 1. 全局设计规范 (Global Design System)

> **复制以下提示词：**
>
> 你是 UI 架构师。请为一款“个人自律管理应用”设计一套 Vue 3 + TypeScript 风格的 Design System。
> **要求：**
> 1. **色彩**：活力蓝主色（专注/效率），配套功能色（绿成功/红错误/黄警告）及 6 级中性灰。
> 2. **组件风格**：圆角 8px-12px，微阴影悬浮卡片，扁平化按钮。
> 3. **排版**：清晰的无衬线字体，基于 4px/8px 栅格系统。
> 4. **响应式**：定义 Mobile/Tablet/Desktop 三端断点。
> 5. **输出**：包含色板、字体规范、按钮/输入框/卡片状态的 UI Kit 概览图。

---

## 2. 核心功能页 (Dashboard & Calendar)

> **复制以下提示词：**
>
> 设计 **仪表盘** 与 **日历打卡** 两个页面，风格为现代 Bento Grid（便当盒布局）。
>
> **页面 A：仪表盘 (Dashboard)**
> *   **KPI 区**：顶部 4 个卡片展示“连续打卡天数（带火苗图标）”、“今日热力值（带趋势箭头）”、“专注时长”、“待办数”。
> *   **图表**：中部展示“近 30 天活动趋势”折线图，极简坐标轴。
> *   **操作**：两个醒目的主按钮“立即打卡”和“开始专注”。
> *   **动态**：底部展示简略的“最近活动时间轴”。
>
> **页面 B：日历打卡 (Calendar)**
> *   **热力图**：顶部展示年度 GitHub 风格绿色热力图，颜色深浅代表活跃度。
> *   **月视图**：日历格子内显示日期，并用 4 色小圆点代表不同事件（打卡/番茄/单词/任务）。
> *   **交互**：点击日期弹出侧边栏详情；设计一个“打卡成功”的弹窗庆祝动效。

## 3. 专注与学习 (Pomodoro & Words)

> **复制以下提示词：**
>
> 设计 **番茄钟** 与 **单词记忆** 页面，强调沉浸感与操作效率。
>
> **页面 A：番茄钟 (Pomodoro)**
> *   **视觉焦点**：中央巨大的渐变色倒计时圆环（Progress Ring），显示 MM:SS。
> *   **控制**：下方悬浮按钮组（开始/暂停/停止）。
> *   **状态**：区分“专注模式”（冷色深邃背景）与“休息模式”（暖色/绿色背景）。
> *   **任务**：顶部显示“当前专注任务”，支持点击切换。
>
> **页面 B：单词复习 (Vocabulary)**
> *   **卡片设计**：拟物化记忆卡片。正面显示单词+音标+发音键；背面显示释义+词性+例句。
> *   **反馈交互**：底部 3 个宽大按钮“忘记（红）”、“模糊（黄）”、“认识（绿）”，按钮上标注下次复习时间（如 10m, 2d）。
> *   **进度**：顶部线性进度条显示今日剩余词数。

## 4. 任务与资源 (Tasks & Books)

> **复制以下提示词：**
>
> 设计 **任务清单** 与 **书籍发现** 页面。
>
> **页面 A：任务清单 (Tasks)**
> *   **列表项**：左侧圆形复选框，中间标题（支持长文折叠）+标签+截止日图标，右侧优先级色条（高红/中黄/低蓝）。
> *   **布局**：支持按“优先级”或“状态”分组。顶部 Tab 切换（待办/已完成）。
> *   **添加**：底部悬浮输入栏，支持快捷指令添加。
> *   **空状态**：插画风格的“无任务”提示。
>
> **页面 B：书籍列表 (Books)**
> *   **网格视图**：展示图书封面（2:3）、书名、作者、星级评分。
> *   **详情交互**：点击卡片弹出模态窗，展示长简介、出版信息及“加入书架”按钮。
> *   **搜索**：顶部大搜索框，支持按标题/作者检索。

---

## 5. 辅助页面 (AI Chat & Login)

> **复制以下提示词：**
>
> 设计 **AI 对话** 与 **登录注册** 页面。
>
> **页面 A：AI 助手 (Chat)**
> *   **对话流**：类 ChatGPT 布局。用户气泡（右侧主色），AI 气泡（左侧灰底带头像）。
> *   **输入区**：底部固定，含输入框、发送键、清除历史键。
> *   **状态**：设计 AI“正在思考”的三点跳动加载态。
>
> **页面 B：登录/注册 (Auth)**
> *   **布局**：桌面端左右分屏（左侧品牌插画，右侧表单）；移动端单列垂直。
> *   **表单**：大尺寸输入框（用户/密码），清晰的错误提示文字。
> *   **行动**：宽度撑满的“登录”主按钮，点击后变 Loading 转圈。
